ISO 15939 Measurement Process Simulator

Student Name: Zeynep Dila Tetik
Student ID: 202328054

This project is a Java Swing desktop application that simulates the ISO/IEC 15939 measurement process.

Requirements:
Java SE 17 or higher

Compilation:
javac *.java

Run:
java Main

Project Features:
- Profile step
- Quality type, mode, and scenario selection
- Measurement plan table
- Data collection and metric score calculation
- Dimension-based weighted average calculation
- Gap analysis
- Radar chart
